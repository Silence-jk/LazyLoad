雷姆，在罗兹沃尔的宅邸中一手担当全部杂务的双胞胎女仆中的妹妹，小时候家人被魔女教所杀，姐姐角被斩断，从而憎恨魔女教，初识昴因其身上有魔女气味不待见昴，之后解开误会被昴拯救，认定昴是她的英雄，一心一意的相信并照顾昴，看似毒舌冷漠，其实内心很坚强，很温柔。

种族：鬼族

武器：流星锤（铁链连着带棘刺的铁球）

能力：家务万能、鬼化（头上长角，战斗力会增强，但也会精神情绪也会受到影响）、魔法、嗅到魔女的气味

身份：露格尼卡王国，处在边境伯罗兹瓦尔宅邸的双胞胎女仆中的妹妹，鬼族后裔

性格：表面恭维内心瞧不起的毒舌担当，维持着宅邸的机能，与笨手笨脚的拉姆不同，在工作上全能。

身高大概150厘米左右，大大的眼睛和桃红色嘴唇，轮廓不太分明的面容显得可爱。发型是齐颈短发，然而头发分界线却有所不同，前留海盖住她的右眼（姐姐是左眼）。一身以黑色为基调的围裙，头上是白色帽饰，除刘海方向、发色和瞳色外和姐姐长得几乎一样，胸部比起姐姐拉姆要更加丰满。

对姐姐拉姆有着绝对的信赖，是个十足的姐控（一直觉得姐姐是最优秀的）。因小时候作为鬼的能力不如姐姐而感到自卑，在姐姐为了救自己而失去了角的力量后为了能保护姐姐，以姐姐为目标努力着

一开始由于昴身上“魔女的香味”以及因为轮回而行动诡 异而对昴充满着敌意和怀疑，甚至在轮回中数次下了杀手，但是在经历了种种事件后开始对昴十分信任，将昴的一切都看在眼里，深深地爱上了昴，性格逐渐开始温和，也变得更加爱笑了。在王选篇的轮回中多次为了昴牺牲了自己，始终陪伴着一度鲁莽行事，不断犯错导致轮回，甚至精神崩溃的昴，如同昴精神支柱一般的存在，也正是因为有雷姆始终如一不离不弃的陪伴，昴才找回了寻找完美结局的动力，也对雷姆产生了感情

宅邸篇中对昴的情感（好感度）随着昴在几次重新读档后的不同表现而不同，曾因昴的行为可疑（不知道昴是读档重来的）怀疑昴是爱蜜莉亚敌对势力或者魔女教派来的间谍而伤害或杀死昴

昴一周目行为自然，雷姆起初对其有疑心，但疑心随着昴在宅邸的笨拙而努力的工作渐渐消夷。

二周目昴行为可疑，第一次来宅邸却十分熟悉宅邸，最后发现昴在半夜居然要去艾米莉亚房间便独断专行地了结了其生命（事后称不忍再看昴受诅咒折磨而下手的，真实原因不知，也可能是因为昴的魔女气息跟可疑行为的原因）

三周目则因为昴选择了当食客而不是佣人，没有好感度，并且昴行为可疑，对其没有丝毫好感，在发现昴出了宅邸后居然监视着宅邸，便判断其绝对是魔女教或者敌对势力间谍，就私自在森林中追杀并拷问昴，后拉姆因看不下去雷姆折磨昴便下手杀了昴。

四周目，昴因为前几周目发现雷姆跟拉姆对其下手而心灰意冷，最后一天躲在贝阿朵莉丝的禁书库里。雷姆只能自己前去村子购买物资（一周目是和男主一起去，男主被魔兽咬伤而受到诅咒死）被魔兽咬伤，受到诅咒，当晚便衰弱死。后来昴在拉姆追杀时终于想通，并立誓“下一次”一定要拯救雷姆跟拉姆后跳崖自杀读档。

五周目，由于昴知道了原先自己昏迷时握住的双手是拉姆跟雷姆的，立誓拯救她们，并发现下咒的是在村里的小狗（魔兽）判断小孩们有危险，想前去村子救小孩们，而拉姆雷姆则一方面对昴存有疑虑一方面也怕小孩有危险，于是雷姆跟昴去村子救人，拉姆则守护宅邸（实际是找贝尔托里丝核实诅咒的事）[2] ，后雷姆跟昴推测小孩在结界破坏掉的树林里，一起进去救人，在逃离时被魔兽袭击，雷姆鬼化后大意差点被魔兽从背后袭击，昴则推开了她自己被魔兽袭击重伤（事后雷姆自责不已，知道那时候自己还能回头再救昴的，却因为他的魔女气息犹豫了）在这次事件中也喜欢上了昴。
